#Installation

Requirements
* Python 3+
* Django 2.x

#Steps For Running This Hello World Django Site On Your Computer

* apt-get install python3 python3-pip
* pip install pipenv
* pip install Django
* git clone https://github.com/aadarshraj231/DjangoApp
* cd DjangoApp-master/
* python manage.py runserver
